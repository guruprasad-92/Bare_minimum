Detailed Description

Purpose

This directory contains highly re-usable code from USB standard and some classes specifications for general use.

Contents

This directory provides two sections of definitions:

  1. Definitions, structures and functions from USB specification, commonly used for all USB classes and projects.
    • core: for USB specification
  2. Definitions, structures and functions from class-specific specification commonly used for class related project ONLY.
    • cdc: for USB Communication Device Class (CDC)
    • hid: for USB Human Interface Device Class (HID)
    • massstorage: for USB Mass Storage Class (MS)
    • audio: for USB Audio Class (AUD)

For more information about what a particular group contains, please refer to its documentation page.
